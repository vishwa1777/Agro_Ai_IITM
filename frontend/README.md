# Agro AI - Field Intelligence Dashboard

Welcome to the **Agro AI** frontend application. This is a modern, responsive web application built with React, Vite, Recharts, and Lucide-React.

## Features Included
- **Dashboard Overview**: Key performance indicators, daily tasks, and weather alerts.
- **Visit Planner**: Detailed route mapping, AI recommendations, and planned visits.
- **AI Recommendations**: Actionable insights for sales reps based on field data.
- **Risk Analyzer**: Comprehensive risk tracking, mitigation suggestions, and risk trend charts.
- **Dynamic Icons**: Premium UI icons powered by `lucide-react`.
- **Responsive Charts**: Interactive data visualizations using `recharts`.

## Getting Started

To run this project locally:

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to `http://localhost:5173/`

## Tech Stack
- React 18
- Vite
- Recharts (Data Visualization)
- Lucide React (Premium Icons)
